# 📚 Advanced Library Management System

A modern, web-based DBMS mini-project built with **Python**, **Flask**, and **SQLite3**. This application provides a clean Graphical User Interface (GUI) to manage a digital library catalog directly from your browser.

---

## 🛠️ Tech Stack
* **Backend:** Python 3.x
* **Framework:** Flask (Web Server)
* **Database:** SQLite3 (Relational Database)
* **Frontend:** HTML5, CSS3 (Modern Responsive Design)

---

## 📂 Project Structure
```text
LibraryProject/
├── app.py              # Main Flask server & routing
├── database.py         # SQL logic & DB connections
├── requirements.txt    # List of Python dependencies
├── README.md           # Project documentation
├── library.db          # SQLite database file
└── templates/          # Folder for UI files
    └── index.html      # Main web interface
✨ Key Features
Full CRUD Operations: Add, view, and delete book records easily.

Instant Search: Filter the collection by Book Title or Author name.

Smart Status Tracking: Visual badges for 'Available' or 'Issued' status.

ID Reset Tool: Integrated feature to wipe data and restart IDs at 1.

🚀 How to Run (On any computer)
Prerequisites
Ensure you have Python installed.

Setup Instructions
Download/Clone the Project:

Bash
git clone [https://github.com/ranvirw18/LibraryProject.git](https://github.com/ranvirw18/LibraryProject.git)
Navigate to the Folder:

Bash
cd LibraryProject
Install Dependencies:

Bash
pip install -r requirements.txt
Launch the Application:

Bash
python app.py
Access the Interface:
Open your browser to: http://127.0.0.1:5000
